
# Networks

This folder contains a curated set of **notes**, **quizzes**, and **labs** designed to support learning in computer networking. The materials are tailored for software engineers (both high-level and low-level), as well as data engineers, focusing on the most relevant networking concepts for these roles.

## Sources & Inspiration

- **Video Lectures:** All content is based on videos from Jeremy's IT Lab, a highly regarded resource for networking education.
- **Curriculum:** Topics are selected from the official curriculum of the University of UC Leuven-Limburg (UCLL), part of Katholieke Universiteit Leuven (KU Leuven).
- **Certification:** The structure and content are aligned with the Cisco CCNA 200-301 exam, but only a selection of topics is included specifically those most relevant for software and data engineers.

## Structure

- **notes/**: In-depth explanations and summaries of networking concepts
- **labs/**: Practical exercises and Packet Tracer files for hands-on learning
- **quizes/**: Self-assessment quizzes to test your understanding *(anwsers + explanation)*
- **images/** and **diagrams/**: Visual aids to support the material *(included in markdown files!)*
