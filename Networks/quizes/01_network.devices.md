# Network Devices Quiz
## Quiz 1
Your company wants to purchase some network hardware to which they can plug the 30 PC's in your departments. Which type of network device is appropriate for this situation?

A) Router
B) Firewall
C) Switch
D) Server

### Anwser
The correct answer is **C**

### Explanation
We have 30 PC's that need to be connected together. This requires a LAN network to be established. A switch is made for the purpose of connecting multiple devices together within a LAN.

## Quiz 2
You recieved a video file from your friends apple iPhone using airdrop. What was his iPhone functioning as in this situation?

A) Router
B) Firewall
C) Switch
D) Server

### Anwser
The correct answer is **D**
### Explanation
In this situation, the iPhone is functioning as a server. A server is a device that provides resources, data, services, or programs to other devices, known as clients, over a network. In this case, the iPhone is providing the video file to your device, acting as a server in the process.

## Quiz 3
What is your computer or smartphone acting as, watching Yemery's IT Lab videos on YouTube?

A) Router
B) Firewall
C) Switch
D) Client
E) Server

### Anwser
The correct answer is **D**

### Explanation
When you are watching videos on YouTube, your computer or smartphone is acting as a client. A client is a device that accesses services or resources provided by a server. In this case, your device is accessing the video content hosted on YouTube's servers, making it a client in this interaction.

## Quiz 4
Your company wants to purchase some network hardware to connect its seperate networks together. What kind of network device is appropriate for this situation?

A) Router
B) Firewall
C) Switch
D) Server
E) Client

### Anwser
The correct answer is **A**

### Explanation
To connect separate networks together, a router is the appropriate network device. A router is designed to route data packets between different networks, allowing them to communicate with each other.

## Quiz 5
Your company wants to upgrade its old network firewall that has been in use for several years to one that provides advanced functions. What kind of firewall should they purchase?

A) host-based firewall
B) next-level firewall
C) next-generation firewall
D) top-layer firewall

### Anwser
The correct answer is **C**

### Explanation
A next-generation firewall (NGFW) is an advanced type of firewall that provides additional features beyond traditional firewalls. These features may include intrusion prevention, application awareness, and advanced threat protection. Upgrading to a next-generation firewall would provide your company with enhanced security capabilities to protect against modern cyber threats.

## Quiz 6
Which of the following best describes a *node* in a network?

A) A device that only sends data  
B) A device that only receives data  
C) Any device that can send or receive data  
D) A device that connects networks together  

### Anwser
The correct answer is **C**

### Explanation
A node in a network is any device that can send or receive data. This includes devices such as computers, smartphones, printers, and servers.

## Quiz 7
You connect 3 laptops together using a single switch. What type of network device is the switch acting as?

A) A device that connects different networks  
B) A device that connects multiple devices within the same network  
C) A device that blocks unwanted traffic  
D) A device that stores files for clients  

### Anwser
The correct answer is **B**

### Explanation
In this scenario, the switch is acting as a device that connects multiple devices within the same network. A switch is designed to connect devices together within a local area network (LAN) and allows them to communicate with each other.

## Quiz 8
Which device is responsible for connecting **different networks** together?

A) Switch  
B) Router  
C) Client  
D) Server  

### Anwser
The correct answer is **B**

### Explanation
A router is responsible for connecting different networks together. It routes data packets between networks, allowing them to communicate with each other. Switches, on the other hand, connect devices within the same network (LAN) but do not connect different networks together.

---

## Quiz 9
Which device is responsible for monitoring and controlling incoming and outgoing traffic based on security rules?

A) Router  
B) Switch  
C) Firewall  
D) Server  

### Anwser
The correct answer is **C**

### Explanation
A firewall is responsible for monitoring and controlling incoming and outgoing network traffic based on predetermined security rules. It acts as a barrier between a trusted internal network and untrusted external networks, such as the internet, to protect against unauthorized access and cyber threats. Routers and switches are primarily responsible for directing traffic, while servers provide services to clients.

## Quiz 10 
Which of the following is considered an **end host (endpoint)** in a network?

A) A device that forwards traffic between networks  
B) A device that provides wireless access  
C) A device that sends or receives data directly (e.g., PC, phone, printer)  
D) A device that filters traffic between networks  

### Anwser
The correct answer is **C**

### Explanation
An end host (endpoint) in a network is a device that sends or receives data directly. Examples of end hosts include personal computers, smartphones, printers, and other devices that users interact with directly. These devices are the source or destination of data in a network.
